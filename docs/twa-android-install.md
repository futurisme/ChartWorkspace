# Android APK Install & Play Guide (ChartWorkspace /game)

This guide explains how to download, install, and play the latest Android build from **GitHub Releases**.

## 1) Download APK
1. Open the repository **Releases** page.
2. Download: `chartworkspace-game-release.apk`.
3. (Optional) Download `chartworkspace-game-release.aab` if you need bundle distribution.

## 2) Allow install from unknown sources
Depending on Android version:
- Android 8+: open the APK once, then allow your browser/file manager to install unknown apps.
- Older Android: enable **Unknown sources** in Security settings.

## 3) Install
1. Tap `chartworkspace-game-release.apk` from Downloads.
2. Tap **Install**.
3. Open app after installation finishes.

## 4) First launch / play
1. Ensure internet connection (TWA requires web content).
2. App opens directly to `/game` scope.
3. Create profile and start playing.

## 5) Update to latest build
- Re-download latest APK from GitHub Releases and install over existing app.
- Data persistence depends on browser/TWA storage behavior.

## Troubleshooting
- **App not installing**: re-check unknown-app permission and storage space.
- **Blank screen/network issue**: verify domain availability and connectivity.
- **Outdated build**: install newest release asset from the latest tag/release.
